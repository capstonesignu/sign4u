# Demo 样例(模型实际输入 → 输出)

> 模型:`exaone-finetuned-v4-cand`(EXAONE-3.5-2.4B,同时会"敬语转换"和"候选挑词")
> 实测于 2026-06-04,可现场复跑。

## A. 敬语转换(词 → 敬语句)

| 输入(词序列) | 模型输出 |
|---|---|
| 나 / 도서관 / 공부 / 하다 | 나는 도서관에서 공부합니다. |
| 머리 / 아프다 / 열 / 나다 | 머리가 아파서 열이 납니다. |
| 약 / 먹다 / 하루 / 세 / 번 | 약을 하루에 세 번 먹습니다. |
| 수술 / 후 / 회복 / 중 | 수술 후 회복 중이라는 것을 알립니다. |

→ 全部输出敬语(합쇼체)✅

## B. 候选挑词(每位置 3 候选 → 挑对的 + 成句)

**模型要从混着干扰词的候选里,挑出正确的那个、再组成敬语句。**(下面每个位置 3 个词,只有 1 个是对的)

| 候选(每位置 3 个,含干扰) | 模型输出 | 参考答案 | 评价 |
|---|---|---|---|
| `나/까지/지표`  `도서관/끝내다/화장품`  `공부/결혼/결막염`  `하다/차갑다/목표` | 나는 도서관에서 공부합니다. | 나는 도서관에서 공부합니다. | ✅ 完全正确 |
| `아빠/창백/장애`  `회사/재능/졸다`  `가다/정돈하다/대기` | 아빠가 회사에 갑니다. | 아빠가 회사에 갑니다. | ✅ 完全正确 |
| `내일/가루약/몇층`  `비/메다/닮다`  `오다/쑤시다/식사`  `우산/혼자/처리`  `필요/이력서/벤치` | 내일 비가 와서 우산이 필요합니다. | 내일 비가 오니 우산이 필요합니다. | ✅ 词全挑对 |
| `동생/물어/협심증`  `학교/순간/비교하다`  `준비/사용/권유`  `하다/교환/밀도` | 동생이 학교에서 준비합니다. | 동생이 학교 준비를 합니다. | ✅ 词全挑对 |
| `김치/나/늘다`  `라면/근무/조금`  `끓이다/정도/이야기`  `먹다/널다/흘리다` | 김치와 라면을 끓입니다. | 나는 라면을 끓여서 먹습니다. | ⚠️ 误选干扰词 김치 |

**6 句里 5 句把正确词从干扰里挑了出来**(其中 2 句和参考答案一字不差),1 句误选了干扰词。
→ 直观证明:**候选机制可行**;小瑕疵(偶尔抓错干扰)后续可用"带把握度的候选"进一步降低。
